# Linking genetic diversity to range shift dynamics

A response to [Oliveira et al. (2026)](https://doi.org/10.1111/ele.70345) in Ecology letters.

## Scripts
- `demography.R`: Explored demographic and phylogenetic determinants of genetic diversity
- `stats-issues.Rmd`: Identifies several statistical issues in the original paper
- `si-figs.R`: Produces a couple figures for the SI
